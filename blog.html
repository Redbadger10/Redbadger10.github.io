Assignment 2 - Styling A Page With CSS

What I found most challenging about this assignment was the non rigid nature of CSS. It felt like many parts of CSS are weirdly intertwined, creating unexpected interactions between objects.
In this assignment, I learned a lot about flex boxes and how they interact with the size of the contents within them. I also learned a lot about spacing elements.
I used a wide range of forums for this assignment as well as the lecture notes from class. I used the website "css-tricks" a lot since it went into detail on how to use flex boxes.
I had to google a lot of things. Despite most of the stuff being covered in lecture, without an experience, there were a lot of things I still didn't fully understand.
